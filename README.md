Pixel Portal Next.js app

```bash
npm install

```

- http://localhost:5173/services/algorithmic-growth
- http://localhost:5173/services/cognitive-seo
- http://localhost:5173/services/omnichannel-architecture
- http://localhost:5173/services/data-engineering

```bash
npm run build
```
